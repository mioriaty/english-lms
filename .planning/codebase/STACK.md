# Tech Stack

**Date:** 2026-04-26

## Core Technologies
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript 5.8
- **UI Library:** React 19

## Database & ORM
- **Database:** PostgreSQL
- **ORM:** Prisma 6.6

## Styling & UI
- **CSS Framework:** Tailwind CSS 4 (via `@tailwindcss/postcss`)
- **Component Library:** Shadcn UI, Radix UI primitives
- **Icons:** Phosphor Icons, Lucide React
- **Animations:** `tw-animate-css`

## State & Data Fetching
- **Client State/Data Fetching:** TanStack React Query v5
- **Form Management:** React Hook Form
- **Validation:** Zod

## Authentication
- **Provider:** NextAuth.js 5.0 (Beta)
- **Encryption:** `bcryptjs`

## Specialized Libraries
- **Rich Text Editor:** Tiptap
- **Drag and Drop:** `@dnd-kit`
- **Data Tables:** TanStack Table v8
- **Media Processing:** `fluent-ffmpeg`, `ffmpeg-static`

## Build & Tooling
- **Package Manager:** pnpm
- **Linter:** ESLint 9 (with Next.js config)
- **Bundler:** Turbopack (configured in `next.config.ts`)
